RustySync - Installation Information
====================================

PREREQUISITES:
- Python 3.8 or higher installed on your system
- Python must be accessible from the command line (python or python3)

INSTALLATION NOTES:
- RustySync will be installed to: Program Files\RustySync
- Python scripts will be located in: Program Files\RustySync\python
- The executable will be at: Program Files\RustySync\bin\rusty-sync.exe

PYTHON DEPENDENCIES:
After installation, you may need to install Python dependencies:
1. Open Command Prompt as Administrator
2. Navigate to: cd "C:\Program Files\RustySync\python"
3. Install requirements: pip install -r requirements.txt

OPTIONAL - ADD TO PATH:
The installer can automatically add RustySync to your system PATH.
This allows you to run 'rusty-sync' from any command prompt.

POST-INSTALLATION:
1. Open a new Command Prompt or PowerShell
2. Run: rusty-sync --help
3. Configure your MinIO server: rusty-sync config

TROUBLESHOOTING:
- If you get "python not found" errors, ensure Python is installed and in PATH
- If commands don't work, try restarting your command prompt
- For permission issues, try running Command Prompt as Administrator

For more information, visit:
https://github.com/TegranGrigorian/rusty-sync
